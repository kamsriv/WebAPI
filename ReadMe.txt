This application is for creating WEB API CRUD Operations with Dapper and SQL Lite.
1. Create a User Registration 
2. Login to the application to see all Registrations
3. Error logging mechanism 

Swagger installation can be done for .net core app
    dotnet add TodoApi.csproj package Swashbuckle.AspNetCore -v 6.2.3
